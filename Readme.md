# Project Compilation and Loading

## Instructions

1. Open a terminal and navigate to the **compiler** folder:
   ```bash
   cd compiler
   ```
2. Run this commands :
    ```bash
    make all
    ```
3. Connect stm32f446re with your device and then run :
    ```bash
    make load
    ```
4. To clean :
    ```bash 
    make clean
    ```
